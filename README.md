# Prerequisites
- There is a docker image that can be pulled and run from docker hub via cvanbroc/fetch_pixels

$ docker pull cvanbroc/fetch_pixels

$ docker run -d -p 5000:5000 cvanbroc/fetch_pixels

# Run test script with pytest

$ pytest test.py
